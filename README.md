# 共享图片仓库 / Shared Images Repository

个人存放图片的仓库，公开图库
Personal image repository, public gallery

## 文件夹结构 / Folder Structure

- `photos/` - 个人照片 / Personal photos
- `screenshots/` - 屏幕截图 / Screenshots  
- `icons/` - 图标文件 / Icon files
- `misc/` - 其他图片 / Miscellaneous images

## 使用说明 / Usage

### 获取图片链接 / Get Image Links

上传图片后，使用以下格式获取公开链接：
After uploading images, use this format to get public links:

```
https://raw.githubusercontent.com/fzxbruce-code/shared-images/main/文件路径/图片名称
```

### 示例 / Example
```
https://raw.githubusercontent.com/fzxbruce-code/shared-images/main/photos/my-photo.jpg
```

## 注意事项 / Notes

- 建议图片文件名使用英文 / Recommend using English for filenames
- 支持常见图片格式：JPG, PNG, GIF, WebP / Supports common formats
- 单个文件大小限制 25MB / Single file size limit: 25MB
